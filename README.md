# 💻 Sistema de Citas Médicas - Frontend

Aplicación web desarrollada en React para la gestión de citas médicas, conectada a una API REST en Spring Boot.

---

## 🚀 Tecnologías utilizadas

- React
- Axios
- Bootstrap
- Vite

---

## ⚙️ Configuración del proyecto

### 1. Clonar repositorio
- git clone https://github.com/fredynofuya/React-API-Citas.git
- cd React-API-Citas
---
### 2. Instalar dependencias 
- npm install
---
### 3. Configurar API
Verifica la URL del backend en los servicios (Axios):
- const API_URL = "http://localhost:8080";
---
### 4. Ejecutar proyecto
- npm run dev
